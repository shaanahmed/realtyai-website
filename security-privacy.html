Security and privacy overview

Local-first by design, practical by default

RealtyAI is designed as a local-first Windows desktop application so your day-to-day listing workflow stays closer to your own machine instead of being forced through an always-online browser stack.

Core security and privacy points

  • Desktop app installed on the user Windows machine
  • Listing workflow and records handled in the desktop environment
  • API keys stored in encrypted form
  • Cloud provider usage only when configured by the user
  • Optional local AI path for privacy-focused workflows

What users should always keep in mind

  • Protect access to your computer and accounts
  • Keep API keys and license credentials private
  • Review AI-generated output before publishing
  • Use human judgment for negotiation and contract decisions

Internet and cloud usage

Some app actions may require internet access:

  • Cloud AI feature calls
  • License activation or status checks
  • App update checks and update downloads

Cloud AI requests are sent only to the provider selected by the user.

Important caution

RealtyAI assists with workflow, drafting, and analysis. It does not replace professional legal, compliance, or brokerage review where required.
