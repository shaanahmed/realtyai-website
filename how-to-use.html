How to use every major part of RealtyAI

This guide helps first-time users test the full workflow in one session, from CSV analysis to export-ready output.

Before you start

  1. Install RealtyAI and confirm the app opens normally.
  2. Complete the first-run wizard.
  3. Download the practice dataset from this page.

What you can complete in one test session

  • Analyze listings from CSV in Comps Analyzer
  • Import records into Portfolio Manager
  • Edit and export in Dashboard
  • Generate single and batch listing copy
  • Use AI Studio market, negotiation, and contract tools

Step-by-step app walkthrough

Follow in order for the smoothest first run.

Step 1: First-run wizard

Open RealtyAI, confirm backend health, complete setup prompts, and continue to Dashboard.

Step 2: Comps Analyzer

Upload the demo CSV and run these sample queries:

  • top 20 cheapest unsold homes in Denver
  • active listings between 500000 and 900000
  • luxury homes over 1000000

Then click Import File to Portfolio.

Step 3: Portfolio Manager

Confirm imported listings, use city filters, edit details, and save updates.

Step 4: Dashboard

Select a listing, edit one detail, and export one listing PDF.

Step 5: Listing Generator

Enter property fields (address, price, beds, baths, square feet, notes), generate copy, then personalize final wording.

Step 6: AI Studio (all 4 parts)

  • Market Snapshot: switch between bars, table, line, heat, histogram, and gauge.
  • Batch Generator: paste rows as address | price | beds | baths | sqft | notes.
  • Negotiation Sandbox: enter property, goal, counterpart position, and context.
  • Contract Risk Highlighter: paste contract excerpt and review flagged clauses.

Step 7: Settings

Configure provider keys if required, validate connection, and run maintenance/update checks.

10-15 minute Try-It challenge

  1. Upload demo CSV in Comps Analyzer.
  2. Run 3 natural-language queries.
  3. Import records to Portfolio.
  4. Edit one listing in Dashboard.
  5. Export one listing PDF.
  6. Generate one single listing copy.
  7. Generate one batch in AI Studio.

Best practices

  • Start with Comps Analyzer, then import to Portfolio.
  • Treat generated copy as a draft and add local detail.
  • Use Market Snapshot before pricing decisions.
  • Use contract risk checks as support, not legal advice.

RealtyAI is a workflow assistant and does not replace legal, financial, brokerage, or compliance review.

Troubleshooting quick notes

  • If app does not load, restart RealtyAI and check backend status.
  • If upload fails, confirm file format is CSV/XLS/XLSX with valid columns.
  • If AI output fails, recheck provider setup in Settings.
