Beginner-first guide

How to use RealtyAI from first launch to advanced workflows

Start simple: configure settings, generate one listing, save it, export it, then move into AI Studio and Portfolio when the basic flow feels easy.

Recommended first-time workflow

Follow these steps in order.

Install and open RealtyAI

Install the Windows app and wait for startup to finish before beginning setup.

Open Settings first

Add a supported cloud AI API key or configure your local AI setup. Check model recommendation, license status, and update status.

Generate your first listing

Use Listing Generator and enter address, price, beds, baths, square feet, and notes. Generate a draft and review for local accuracy.

Save and manage in Dashboard

Search, filter, edit, restore, delete, and export from one page. Use Dashboard as your daily control center.

Export a polished PDF

After final review, export a professional PDF for internal use, client sharing, or listing presentation.

Use AI Studio for deeper support

Review market snapshots, run batch listing generation, prepare negotiation talking points, and highlight risky contract wording.

Use Portfolio for larger workloads

Compare grouped listings and organize work by city or area when handling multiple properties.

Beginner guidance

  • Start with one listing, not many
  • Learn Generator and Dashboard first
  • Review all AI output before client use
  • Move to AI Studio only after core flow is comfortable

RealtyAI is a workflow assistant and does not replace legal, compliance, brokerage, or professional judgment requirements.
