Short FAQ

Is RealtyAI a web app?

No. RealtyAI is a Windows desktop app.

Do I need internet?

Some tasks can run with local AI, but cloud AI requests, updates, and some licensing actions may require internet access.

Do I need an API key?

If you use cloud AI providers, yes. If you use a supported local AI setup, some workflows may not require cloud keys.

Is data stored online?

RealtyAI is designed for local-first workflows. Cloud requests only go to providers you choose to connect.

Can non-technical users use RealtyAI?

Yes. The workflow is designed so users can start simple with listing generation and learn advanced features gradually.

Is contract screening legal advice?

No. It is a support tool for highlighting areas that may need closer professional review.

Need direct help?

Support can assist with installation, licensing, API setup, local AI setup, updates, and workflow questions.
